Standalone Examples » Pie Chart Canvas Size

This example shows how to specify a size for your pie chart canvas. 

var pie = new d3pie("pie", {
  size: {
    canvasHeight: 120,
    canvasWidth: 140
  },
  labels: {
    outer: {
      format: "none"
    }
  },
  data: {
    content: [
      { label: "JavaScript", value: 241 },
      { label: "Ruby", value: 525 },
      { label: "Java", value: 150 }
    ]
  }
});
